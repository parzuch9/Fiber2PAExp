EMCCDcontrol.ipy: the code that was used to control the final fiber experiment
includes functions for running C2PA or E2PA measurements as well as concentration measurements using the LED
there is also an EMCCD alignment function

EMCCDsettings.py: the code used to chose the EMCCD settings
If you have an anticipated signal level and background level spread over a certain number of pixels,
you can guess which settings are the most advantageous

sCMOScontrol2.ipy: the code used to acquire single images from the sCMOS and to align it
